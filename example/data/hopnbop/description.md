> Boing, boing, splat!

Hop 'n Bop is an online, cross-platform, competitive 2D arena brawler.

This game is based-off a true work of art from the 90s called Jump 'n Bump--which you can also play online at (jumpnbump.net)[https://jumpnbump.net/]!

Levi used Hop 'n Bop as a simple game to develop and test a [custom rollback netcode library](https://github.com/SnoringCatGames/godot-rollback-netcode), which features instant input response and server-authoritative gameplay.

### 🛠️ Tools

*   **[Godot](https://godotengine.org)**: Game engine.
*   **[Aseprite](https://www.aseprite.org/):** Pixel-art image editor.
*   **[FamiStudio](https://famistudio.org/)**: Music tracker.
*   **[BFXR](https://www.bfxr.net/)**: 8-bit SFX generator.
*   **[VSCode](https://code.visualstudio.com/)**: IDE.
*   **[Claude Code](https://code.claude.com)**: Code assist.

_All art, music, and sound effects were made without the use of AI._
